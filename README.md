# Game Launcher - JavaFX

This project is a **Game Launcher** developed using **JavaFX**.  
It includes three classic mini-games:
- **Flappy Bird**
- **Snake**
- **XO**

The launcher provides an easy-to-use interface to select and launch any of the games.
